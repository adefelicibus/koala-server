Koala Server


This is an open server to anyone can use for running tools and workflows on real-world data. However, the computational resources are limited. We have plans to improve the infrastructure in order to useful production server. Please, use it carefully because this server is alpha version.


This is a personalized instance of the Galaxy Framework



As it is an open server, you can use the server without logging. Although, to use the workflow and published data, you must log in. To create an account, register under "User->Register".
With an account, login under "User->Login".
