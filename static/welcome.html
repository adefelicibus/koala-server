Koala Server


This is an open server to anyone can use for running tools and workflows on real-world data. However, the computational resources are limited. We have plans to improve the infrastructure in order to create a useful server. Please, use it carefully because this server is alpha version.


This is a personalized instance of the Galaxy Framework



Login
As it is an open server, you can use the server without logging. Although, to use workflow and shared resources, you must log in. To create an account, register under "User->Register".
With an account, log in under "User->Login".
Workflows available
Koala Server has a set of generic workflows available. You can import, modify or run them by accessing the Shared Data - Published Workflows or by this link.
The workflows Ranking of solutions by Sort by Front Dominance and Comparison of predictions by RMSD, GDT-TS and TM-Score are the two most completed workflows built on Koala Server to perform predictions with all the algorithms available and analysis with different methods.
Ranking of solutions by Sort by Front Dominance: This workflow has 16 steps to run. This runs all the predictor algorithms available on Koala using the same amino acids sequence. Once all the algorithms are finished, their results are sent to the method Sort by Front Dominance to create a ranking with the best solutions according to a combination of two objectives. The final result is an HTML report.
Comparison of predictions by RMSD, GDT-TS and TM-Score: This workflow has 19 steps to run. This workflow also runs all the predictor algorithms available on Koala. Besides the amino acid sequence, this workflow recieves a PDB file to be used as a reference to the metrics RMSD, GDT-TS and TM-Score. All the results generated by the algorithms are sent to these metrics. The final result is an HTML report.
